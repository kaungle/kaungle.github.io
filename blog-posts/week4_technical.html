Enumerables!

The #cycle Method!


What is Enumerable?

Becasue Ruby has modules that serve as containers for methods and constants, code can be shared between multiple unrelated classes with Enumerable. An 'each' method defines the parameters with which the iteration of Enmerable acts upon a collection, and gives you access to a whole bunch of awesome methods! Some of the instance methods available to Enumerable can even create infinite loops through a collection.


What does Enumerable#cycle Do?

This instance method can be passed a numerical argument and will totally iterate through a collection that number of times. If '#cycle()' is given, the loop is infinite. The method provides a cleaner way to acomplish this iteration without using the 'times' or 'each' methods.


When/How To Use Enumerable#cycle?

Given an array or string of data, '#cycle' can loop through a collection a specified number of times. It cannot, however, tell the number of elements to cycle through without an Enumerator. Nevertheless, this method can be used to assert control over which objects to iterate through in a collection. It is like using '#each' in an infinite loop.
DON'T USE #CYCLE {}! This will create a real deal infinite loop in your code.
This method is useful, though, in providing a faster and cleaner way to loop through a collection.
